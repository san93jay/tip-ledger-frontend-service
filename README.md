Prerequisites to Install

Node.js

Package Manager

npm comes with Node.js.

verify version node -v npm -v

Git (optional but recommended)

To clone the repository and manage version control.

Code Editor

Visual Studio Code editor. Steps to Run the Project

Clone the repository

goto project director by -> cd (project-folder)

Install dependencies by npm install

Start the development server npm run start -> for Local run

Open in browser

By default http://localhost:30001/login

<Note> -> Before starting frontend application , start you backend application and RabbitMQ server
